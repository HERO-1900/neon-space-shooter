# Neon Space Shooter 🚀

一款现代化的 HTML5 Canvas 太空射击游戏。

## 🎮 游戏特色

- **流畅的 60fps 游戏体验**
- **霓虹赛博朋克视觉风格**
- **多种敌人类型**：普通、快速、坦克、Boss
- **粒子爆炸特效**
- **道具系统**：生命值恢复、武器升级
- **等级系统**：随着分数提升难度递增
- **响应式控制**

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| ← ↑ ↓ → | 移动飞船 |
| 空格键 | 射击 |

## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/HERO-1900/neon-space-shooter.git
cd neon-space-shooter
```

2. 在浏览器中打开 `index.html`

或使用本地服务器：
```bash
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 🏆 高分榜系统

游戏使用 localStorage 持久化存储高分记录：

- **自动保存**: 游戏结束时自动检测是否为新高分
- **名字输入**: 支持自定义玩家名字（最多10字符，自动大写）
- **排行榜**: 显示前10名高分，带排名奖牌（🥇🥈🥉）
- **当前最高分**: 游戏界面实时显示当前最高分
- **快捷操作**: 支持 Enter 键快速提交名字

### 高分榜界面

- 从主菜单点击 **🏆 HIGH SCORES** 查看排行榜
- 游戏结束时点击 **🏆 HIGH SCORES** 查看最新排名
- 新高分时自动弹出输入界面

运行测试：
```bash
npm test
```

## 🎵 音效系统

游戏使用 Web Audio API 合成音效，无需外部音频文件：

- **射击音效**: 激光射击声音
- **爆炸音效**: 小型爆炸和 Boss 大型爆炸
- **受伤音效**: 玩家飞船被击中
- **道具音效**: 收集道具时的 C 大调琶音
- **游戏结束**: 悲伤的 A 小调下行音阶
- **背景音乐**: 动态低音贝斯线 + C 大调琶音循环

点击右上角 **AUDIO ON/OFF** 按钮可切换音效。

运行测试：
```bash
npm test
```
