# Neon Space Shooter 🚀

一款现代化的 HTML5 Canvas 太空射击游戏。

## 🎮 游戏特色

- **流畅的 60fps 游戏体验**
- **霓虹赛博朋克视觉风格**
- **多种敌人类型**：普通、快速、坦克、Boss
- **粒子爆炸特效**
- **道具系统**：生命值恢复、武器升级
- **等级系统**：随着分数提升难度递增
- **响应式控制**

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| ← ↑ ↓ → | 移动飞船 |
| 空格键 | 射击 |

## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/HERO-1900/neon-space-shooter.git
cd neon-space-shooter
```

2. 在浏览器中打开 `index.html`

或使用本地服务器：
```bash
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```
